# Excel Sales & Profit Dashboard
## Project Overview
This project analyzes sales data using Microsoft Excel.
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sum of sales
- Sales by State
- Sales by Segment
- Profit by Category
- Top 10 Products by Sales
## Key Insights
- The Technology category generates the highest total sales compared to other categories.
- Technology category generates the highest profit among all categories.
- The top 10 products contributes a major portion of total sales.
- Consumer segment contributes the most revenue.
- England generates the highest sales.
